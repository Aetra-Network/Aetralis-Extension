# Aetralis Language

VS Code support for the Aetralis `.atlx` language used to develop smart contracts on the Aetra blockchain.

## Features

- syntax highlighting for the language surface used in `counter_should_be.atlx`
- diagnostics for unmatched brackets, malformed annotations, broken strings, comments, and unknown identifiers
- autocomplete for annotations, built-ins, declared symbols, and member access
- hover help for keywords, annotations, built-ins, and symbols
- document symbols and semantic highlighting
- snippets for common Aetralis constructs

## Scope

The extension intentionally keeps the highlighted keyword set narrow and matches the language surface demonstrated in `counter_should_be.atlx`.
That keeps the coloring predictable and avoids extra keywords that are not part of the current language model.
