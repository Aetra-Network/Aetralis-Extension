# Aetralis Language

Syntax highlighting, autocompletion, hover help, diagnostics, and snippets for the Aetralis language.

## Features

- syntax highlighting for language keywords, types, names, strings, and operators
- autocompletion for annotations, built-ins, symbols, and message builders
- hover help for annotations, keywords, and built-in helpers
- diagnostics for malformed syntax, reserved names, and unknown identifiers
- document symbols and semantic highlighting
- snippets for common Aetralis constructs
